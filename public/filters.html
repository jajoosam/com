Filter Types:


Most things here are case sensitive, and whitespace sensitive.

  • from:ElonMusk | This specifies which account the tweet should be from
  • min_retweets:number-goes here | Append this to a filter, to only see tweets which have at least x retweets
  • min_faves:number-goes here | Append this to a filter, to only see tweets which have at least x favorites
  • url.com/yourownurl | See only tweets which mention a particular URL
  • filter:verified | Append this to any filter to only show the tweets which come from verified accounts
  • filter:news | Append this to any filter to only show the tweets which link to a news website
  • filter:safe | Append this to any fiter to excluse sensitive/adult content
  • filter:images | Append this to any filter to see only tweets which contain images
  • -RT | Append this to any filter to exculde retweets which match the filter
  • #hashtag | Shows tweets with a particular hashtag
  • near:place within:10mi | Filter tweets which come within a particular radius from a place
